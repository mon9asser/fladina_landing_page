Installation And Folder Structure ..
(A) Folder Structure
When you Extract downloaded archive from ThemeForest.net you'll get folder containing 2 folders.
  • 1- HTML This is Main File for Template
  • 2- Documentation This is a file included in installation method and other info , will help you to do any changes in colors and elements
----------------------------------------------------------------------------------------
Inside HTML Folder you will find the following :-
  • 1- assets/ This File include all the follwing
    --(A)- css
    --(B)- fonts
    --(C)- images
    --(A)- js
  • 2- Php/
  • 3- All Dimeos.html
(B) Installation
it is very easy to upload the theme files to your server by the following steps :-
  • 1- Go to HTML Folder of our theme then rename html files to what you like exmple : index.html , main.html
    2- Open (FTP) Program and connect to your server
    3- By (FTP) Program Select Theme Files then upload it
    4- Congratulations , you're done it , ( Pleas feel a free to contact us for any details )

Colors and Themes
A- ( How to change Style File ) To change style file there are many files each one has a different colors , the following show you each one with each color
Theme-blue-family-2.css
Theme-yellow-family-1.css

Theme-purple-family-1.css

after select file.css you should search about this line <!-- Theme colors --> you will find the following code , Add your file path in double quotation 
                    
                      <link href="...." rel="stylesheet">
B- ( How to create new style and change the Colors ) it's easy to create new stylesheet file just do the following steps
-- 1 create fileName.css in this dir assets/css/ then go to this file default.css and open it
-- 2 copy all classes and id inside this file default.css
-- 3 Pest them in your new file.css
-- 4 Open This website flatuicolorpicker and chose the colors as you like (Hex or RGB) then copy Hex value
-- 5 Go to your file.css again to change the colors :-
++++ Backgrounds , Borders , banners : serach about this line /*--- BG COLORS ---*/ and Replace all with your new hex color
++++ Font Colors : serach about this attribute color and give it new hex color
++++ Image Backgrounds : serach about this attribute background-image and put your image path here url(... Image Path here ...)
-- 5 after complete all this steps you should includ this new file.css in html like the code bellow 
                         
                           <link href="assets/css/yourFile.css" rel="stylesheet">
